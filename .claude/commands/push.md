Push the current changes to GitHub by doing the following steps in order:

1. Run `git status` to see what has changed
2. Run `git diff` to review the changes
3. Run `git add -A` to stage all changes
4. Ask the user for a commit message, or if they provided one as an argument use that (via $ARGUMENTS)
5. Commit with the message provided
6. Run `git push` to push to the remote repository
7. Report the result — confirm success or show any errors
